# 📚 NovelNest

### 📖 Where every book opens a new world — explore, discover, and read with NovelNest.

NovelNest is a front-end online bookstore website developed using HTML, CSS, and JavaScript. It provides an engaging platform for readers to explore books across multiple genres, search categories, create accounts, sign in, and manage their shopping cart.

## ✨ Features

- 🔍 Smart book and category search
- 📖 Browse books by genre
  - Fiction
  - Non-Fiction
  - Psychology
  - Love
  - Health & Wellness
  - Poetry
  - Biography
- 🛒 Add-to-Cart functionality using Local Storage
- 👤 User Sign Up and Sign In pages
- 👋 Personalized welcome message after login
- 🎨 Interactive UI with hover effects and animations
- 📱 Responsive navigation bar
- 📚 New Arrivals section
- 🔗 Footer with quick navigation links and social media icons

## 🚀 How to Run

1. Download or clone the repository.
2. Open `NovelNest.html` in any web browser.
3. Create an account using the Sign Up page.
4. Sign in and start exploring books.

## 💡 Functionalities

### User Authentication
- Users can create an account.
- Credentials are stored using Local Storage.
- Sign In validates stored credentials.

### Search System
- Real-time category suggestions.
- Direct navigation to relevant genre pages.

### Shopping Cart
- Add books to cart.
- Cart data stored in Local Storage.
- View and manage selected books.
